"I have built a MERN stack e-commerce project using HTML, CSS/Tailwind, JavaScript, and React for the frontend, and Node.js, Express.js, and MongoDB for the backend. The frontend runs with npm run dev and the backend with nodemon index.js. The platform includes categories like Fashion, Dresses (Men & Women), Vehicles, Electronics, and Fruits. It features secure authentication using Auth Tokens and password protection, integrated with a Fake API."
